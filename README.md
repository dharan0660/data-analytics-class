# Data Analytics Class — Structured / Semi-Structured / Unstructured Datasets

This repo contains three medium-sized, publicly available datasets, organized by data type.

## 📁 structured/
**titanic.csv** — Classic Titanic passenger dataset (rows/columns, clean tabular CSV).
Source: [datasciencedojo/datasets](https://github.com/datasciencedojo/datasets)

## 📁 semi-structured-data/
**cities.json** — Nested JSON dataset of world cities (country, name, lat/lng).
Source: [lutangar/cities.json](https://github.com/lutangar/cities.json)

## 📁 unstructured/
### text/
**tinyshakespeare.txt** — Raw text corpus of Shakespeare's works (public domain).
Source: [karpathy/char-rnn](https://github.com/karpathy/char-rnn)

### images/
Sample public-domain test images (JPG/PNG) commonly used in computer vision:
`lena.jpg`, `baboon.jpg`, `fruits.jpg`, `messi5.jpg`, `home.jpg`, `smarties.png`
Source: [opencv/opencv samples](https://github.com/opencv/opencv/tree/master/samples/data)

### video/
**vtest.mp4** — Short sample video clip (converted from OpenCV's test AVI to compact MP4).
Source: [opencv/opencv samples](https://github.com/opencv/opencv/tree/master/samples/data)
